# handmade-store-front

## セットアップ

```
npm install
```

### vscode の拡張

```
code --install-extension dbaeumer.vscode-eslint

code --install-extension msjsdiag.debugger-for-chrome
code --install-extension octref.vetur
code --install-extension esbenp.prettier-vscode
```

### 起動

```
npn run serve
```
